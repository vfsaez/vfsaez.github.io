# Victor Saez - Portfolio

Professional portfolio website built with React, TypeScript, and Vite.

## Live Site

Visit the portfolio at: [https://vfsaez.github.io](https://vfsaez.github.io)

## Development

### Prerequisites
- Node.js 20 or higher
- npm

### Installation

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Deployment

This site is automatically deployed to GitHub Pages using GitHub Actions. Every push to the `main` branch triggers a new deployment.

## Tech Stack

- **React 18** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **GitHub Pages** - Hosting
- **GitHub Actions** - CI/CD
